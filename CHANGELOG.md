Toggle Tunlr Changelog
======================


v2.0.0 [2012-07-29]
--------------------

- Major refactoring
- Makes it possible to explicitly set the Tunlr state
  - `tunlr on`
  - `tunlr off`
- Makes it possible to check the current state
  - `tunlr status`
- Bug fix: the OS X version no longer gets piped into a file

v1.0.0 [2012-07-29]
--------------------

- Initial working version
